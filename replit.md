# حافِظي — تطبيق إسلامي شامل

## نبذة
تطبيق ويب إسلامي متكامل باللغة العربية (RTL) يساعد المسلم في رحلته مع القرآن الكريم والأذكار والأحاديث النبوية والأدعية. اللون الأساسي: البرغندي الداكن مع الأخضر الزمردي (teal).

## البنية التقنية
- **Frontend**: React + Vite + TypeScript (artifacts/quran-tracker)
- **Backend**: Express API (artifacts/api-server)
- **State**: TanStack Query
- **التوجيه**: Wouter
- **Style**: Tailwind CSS + shadcn/ui
- **DB**: PostgreSQL عبر DATABASE_URL

## الصفحات (6 صفحات)

### 1. الرئيسية (`/`)
- دائرة تقدم الحفظ
- إحصائيات (صفح محفوظة، سور مكتملة)
- بطاقات التنقل للأقسام الستة
- زر إعادة ضبط البيانات

### 2. المصحف (`/quran`)
- قائمة السور الـ 114 (مصحف المدينة، 604 صفحات)
- startPage لكل سورة (الأرقام الحقيقية من المصحف)
- عرض موسّع: صناديق الصفحات مرقّمة بأرقام المصحف الفعلية (ص1 ← ص604)
- حفظ تقدم كل صفحة مع checkmarks في قاعدة البيانات

### 3. الأذكار (`/adhkar`)
- تبويب صباح/مساء
- 18 ذكر صباحي + 17 ذكر مسائي من حصن المسلم
- عداد لكل ذكر مع checkmarks
- الأذكار كاملة بنصها الصحيح

### 4. التحصينات (`/tahseenat`)
- 10 تحصينات مع ملاحظات وفضائل
- نصوص كاملة (آية الكرسي، المعوذتان، سيد الاستغفار...)

### 5. الأدعية (`/adiyah`)
- 11 تصنيفاً يشمل: ختم القرآن، الكرب، الاستخارة، الرزق، الشفاء، السفر، المسجد، الطعام، النوم، الأدعية القرآنية، التوبة
- accordion تفاعلي

### 6. الأحاديث (`/ahadith`) — الأربعون النووية
- كاملة 42 حديثاً بنصها العربي الصحيح
- الراوي والمصدر والفائدة لكل حديث
- بحث نصي + accordion
